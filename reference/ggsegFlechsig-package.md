# ggsegFlechsig: Flechsig Myelogenetic Atlas for the 'ggseg' Ecosystem

Flechsig myelogenetic cortical atlas for the 'ggseg' ecosystem. Provides
a unified 'ggseg_atlas' object with both 2D polygon geometry and 3D
vertex indices, for use with 'ggseg' and 'ggseg3d'. Based on the
supplementary materials of Pijnenburg et al. (2021).
